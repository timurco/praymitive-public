## Material Mastery with Smooth Booleans

Every material setting blends perfectly through Smooth Booleans, letting you combine and transition materials effortlessly:

- **Diffuse** — precise color and intensity control
- **Specular** — dial in reflections, from color to shininess
- **Glossiness** — that perfect reflective touch
- **Rim Light Options** — unique edge glow, even colorized using object normals

---

## Modifier Mastery: Unleash Dynamic Forms & Motion

With PrayMitive's modifiers, elevate your designs to unprecedented levels:

- **Waves modifier** — create wavy serpents, blended organically
- **Ripples modifier** — billowing flag or shimmering water surface
- **Twist modifier** — intertwined forms full of life, paired with undulating waves
- **Bend modifier** — mold a wriggling worm or a smoothly curved object

And the best part? **Every transformation can be animated**, morphing static artistry into dynamic motion.

## Installation

1. Purchase PrayMitive from [aescripts.com](https://aescripts.com/praymitive/)
2. Download the plugin
3. Install using aescripts + aeplugins manager or manually copy to your After Effects plugins folder
4. Restart After Effects
